This is an imageboard viewer primarily aimed at mobile devices, providing an image search, display, and extending it with a javascript image gallery. It uses the built in APIs of targeted imageboards to deliver images to users. AJAX requests are used and sent to a custom server that I have set up which does some of the image processing to present them in a better format on the viewer.

This particular version is pointed at e926.net and is locked to that site.

Released for: Android, Windows-Any
